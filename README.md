# TaskFlow

TaskFlow — веб-приложение для управления задачами, разработанное на Django.

## Возможности

- Регистрация и авторизация пользователей
- Создание, просмотр, изменение и удаление задач
- Изменение статуса задач:
  - Не начато
  - В процессе
  - Завершено
- Создание и управление личными категориями
- Изоляция данных пользователей:
  - каждый пользователь видит только свои задачи;
  - каждый пользователь работает только со своими категориями.

## Технологии

- Python
- Django
- SQLite
- HTML
- CSS

## Установка

Клонировать репозиторий:

```bash
git clone <repository_url>
```

Перейти в директорию проекта:
```bash
cd TaskFlow
```

Создать виртуальное окружение:
```bash
python -m venv .venv
```

Активировать виртуальное окружение:
Windows:
```bash
.venv\Scripts\activate
```

Установить зависимости:
```bash
pip install -r requirements.txt
```

Выполнить миграции:
```bash
python manage.py migrate
```

Создать суперпользователя (опционально):
```bash
python manage.py createsuperuser
```

Запустить сервер:
```bash
python manage.py runserver
```

Приложение будет доступно:
```text
http://127.0.0.1:8000/
```

Структура проекта
```text
TaskFlow/
├── config/              # настройки проекта Django
├── tasks/               # задачи и категории
├── users/               # регистрация и авторизация
├── manage.py
└── requirements.txt
```
